# subagent-strategy

サブエージェントを活用した並列実行・調査委譲・コンテキスト管理の戦略を定義するスキルです。

## 使い方

```
/subagent-strategy
```

引数は不要です。調査・並列処理・大規模変更に着手する前に呼び出します。

## いつ使うか

- 複雑な問題への対処や調査・探索タスク
- 複数の選択肢・ライブラリの比較検討
- 大規模なリファクタリングや複数ファイルの並列解析
- 「調べて」「比較して」「並列で」「複数のアプローチを試して」といったリクエスト
- メインのコンテキストが肥大化しそうな場面

## 動作

1. **委譲判断**: 調査・並列分析・独立した作業単位をサブエージェントへ委譲
2. **1エージェント1タスク**: 焦点を絞った単一タスクをサブエージェントに与える
3. **結果の統合**: 複数サブエージェントの結果をメインコンテキストで比較・統合

## インストール

```bash
# グローバル（シンボリックリンク）
ln -s ~/path/to/claude-code-tools/skills/meta/subagent-strategy ~/.claude/skills/subagent-strategy

# プロジェクト（コピー）
cp -r ~/path/to/claude-code-tools/skills/meta/subagent-strategy .claude/skills/
```
