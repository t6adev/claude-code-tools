# frontend-design

## 概要

汎用的なAI生成UIの見た目を避け、個性的で本番品質のフロントエンドデザインを生成するためのガイダンスプラグイン。大胆な美学・タイポグラフィ・カラーパレット・アニメーションの指針を提供します。

## ソース

- Repository: https://github.com/anthropics/claude-code
- Plugin path: `plugins/frontend-design/`
- Author: Prithvi Rajasekaran & Alexander Bricken (Anthropic)

## 採用理由

「AIらしいデザイン」に陥らず、本番品質のUIを生成したい場合に有効。Frontend Aesthetics Cookbook に基づいた具体的なデザイン指針を提供する。

## インストール

```bash
claude plugin marketplace add anthropics/claude-code
claude plugin install anthropics/claude-code/plugins/frontend-design
```

## 提供機能

- 独自性のあるビジュアル特性のガイダンス
- タイポグラフィ選択の指針
- カラーパレット設計のベストプラクティス
- アニメーション・インタラクションの指針
- 汎用AI美学（白背景・ありきたりなカード・過度なシャドウ等）を避けるルール
